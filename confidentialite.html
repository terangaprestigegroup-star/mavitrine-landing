Politique de confidentialité

Mentions légales CGV

1. Données collectées

Pour fonctionner, MaVitrine.sn collecte les données suivantes :

  • Nom complet du commerçant
  • Numéro de téléphone (utilisé pour la connexion et les notifications)
  • Nom de la boutique, ville, secteur d'activité
  • Code PIN d'accès (stocké de manière sécurisée)
  • Produits ajoutés (nom, prix, description, image)
  • Commandes reçues via la boutique (nom et téléphone du client final, si fournis)
  • Statistiques d'utilisation (nombre de vues de la boutique, clics WhatsApp)

2. Utilisation des données

Les données sont utilisées exclusivement pour :

  • Permettre la création et le fonctionnement de la boutique en ligne
  • Communiquer avec le commerçant via WhatsApp (confirmation de paiement, support)
  • Générer des statistiques d'utilisation visibles par le commerçant lui-même
  • Améliorer le Service (analyse anonymisée d'usage)

MaVitrine.sn ne vend ni ne partage les données personnelles à des tiers à des fins commerciales.

3. Hébergement et sécurité

Les données sont stockées sur des serveurs Supabase (PostgreSQL). L'accès à l'API backend nécessite une authentification par jeton sécurisé (JWT), avec une durée de validité limitée à 7 jours. Nous travaillons continuellement à renforcer la sécurité de stockage des données sensibles, notamment le code PIN.

4. Assistant Vente IA

Lorsqu'un commerçant utilise l'Assistant Vente IA, les informations saisies (nom du produit, prix, cible) sont transmises à un service tiers d'intelligence artificielle (Anthropic) pour générer des messages de vente. Aucune donnée personnelle du client final n'est transmise dans ce processus.

5. Droits de l'Utilisateur

Conformément à la loi sénégalaise sur la protection des données personnelles (loi n° 2008-12), chaque commerçant peut :

  • Demander l'accès à ses données personnelles
  • Demander la correction de données inexactes
  • Demander la suppression de son compte et de ses données

Ces demandes peuvent être effectuées via WhatsApp depuis la page d'accueil.

6. Conservation des données

Les données sont conservées tant que le compte du commerçant est actif. En cas de suppression du compte, les données sont effacées dans un délai de 30 jours, sauf obligation légale de conservation plus longue.

7. Cookies et traceurs

MaVitrine.sn utilise uniquement des données de connexion stockées localement sur l'appareil de l'Utilisateur (localStorage) pour maintenir la session connectée. Aucun cookie publicitaire ou traceur tiers n'est utilisé.

8. Contact

Pour toute question relative à vos données personnelles, contactez-nous via WhatsApp depuis la page d'accueil de MaVitrine.sn.
